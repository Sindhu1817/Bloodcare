Transforming Blood Donation Through Technology

A modern full-stack web application connecting donors, hospitals, and patients quickly and securely.

Project Overview

The Blood Donation Management System is designed to reduce the time required to find blood during emergencies. Users can register as donors, search blood availability, and send urgent requests through a secure digital platform.

The system uses a responsive frontend built with HTML, CSS, and JavaScript, while Flask handles backend logic and MySQL securely manages donor data. The platform ensures fast matching between donors and patients.

Workflow: User Request → Backend Processing → Database Match → Instant Response. This architecture ensures reliability, speed, and scalability.
